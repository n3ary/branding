1. Logo

Primary mark

A stylized location pin — the most direct metaphor for what neary does: locate, transit, near.

Saturated indigo background, bold white mark, indigo "stop dot" punched out at the centre.

Asymmetric by construction: the pin's pointed base sits off-centre to the left.

Variants

Primary — saturated indigo. Default for everything.

Maskable — for Android adaptive icons (safe-area pulled in).

Monochrome — single-ink black on transparent.

Wordmarks

neary wordmark — product pages, README headers.

n3ary wordmark — org-level docs, the n3ary.com masthead.

Wordmarks on dark

Use the *-wordmark-light.svg variants on dark surfaces. Same mark, white text fill — readable on slate-900 and dim cards. The dark-text wordmarks above will disappear on these backgrounds.

3. Typography

System sans-serif stack

The wordmark and brand type use the system font stack for portability — no web-font loading, works offline.

6. Don't

  • ❌ Rotate the mark
  • ❌ Add a stroke / outline
  • ❌ Use a gradient or drop shadow
  • ❌ Recolor (other than the official palette)
  • ❌ Use green anywhere except the live GPS vehicle accent
  • ❌ Place on a background that loses contrast
  • ❌ Stretch non-uniformly
  • ❌ Add elements behind or around the mark
  • ❌ Use the monochrome variant on color
